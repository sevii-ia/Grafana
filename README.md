# Grafana

Educational repository dedicated to learning and practicing monitoring dashboards, observability, and Grafana configuration.

This repository contains study materials, dashboard examples, monitoring experiments, and practical tasks related to Grafana and infrastructure visualization.

---

## 📚 About

The project was created for educational purposes and includes:

- Grafana dashboard experiments
- Monitoring visualization practice
- Metrics and observability learning
- Data source configuration examples
- Infrastructure monitoring exercises
- DevOps and system administration practice

This repository is intended for learning and experimentation rather than production use.

---

## 🛠️ Technologies & Tools

- Grafana
- Prometheus
- Docker
- Linux
- Node Exporter
- InfluxDB
- MySQL / PostgreSQL
- Networking tools

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/sevii-ia/Grafana.git
cd Grafana
````

Run the project:

```bash
grafana.sh
```

Open in browser:

```txt
http://localhost:3000
```
